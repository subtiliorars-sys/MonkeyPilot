🤖 Bot-Detection Playground

Test the Chrome Agent's bot-evasion features against common form protection methods.

Red-Teaming Security Rules Active:

  • Honeypot Trap: Hidden input field must remain empty (bots often auto-fill all inputs).
  • Coordinate Check: Form submission clicks must have valid screen coordinates (clientX / clientY cannot be 0).
  • Keystroke Tracker: Text inputs must be typed key-by-key (keydown, input, keyup events are tracked). Fails if values are pasted all at once.
  • Speed Limit Check: Typing speed is tracked. Fails if typing occurs instantaneously (must take > 0.5 seconds).
